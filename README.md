# Shopping Web-App

## Getting Started

Before you must create a new environment

Windows

```bash
py -m venv .venv
```

Unix/MacOS

```bash
python -m venv .venv
```

Active environment

Windows

```bash
.venv\Scripts\activate.bat
```

Unix/MacOS

```bash
source .venv/bin/activate
```

Install `requirements.txt` file

```bash
pip install requirements.txt
```

Run application

```bash
py manage.py runserver
```
